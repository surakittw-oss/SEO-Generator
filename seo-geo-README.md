# SEO · GEO · AEO Generator 2026

สร้าง SEO / GEO / AEO metadata ครบชุด 5 รูปแบบ จากเนื้อหาบทความ

**0 token · ไม่ใช้ API · รันใน browser 100%**

---

## ฟีเจอร์

- วางเนื้อหาบทความ → กด Generate → ได้ 5 รูปแบบทันที
- แต่ละรูปแบบมีครบ: SEO Keyword, Meta Description, SEO Title, Facebook Title, Twitter (X), URL Slug, Alt Text
- Copy แยกแต่ละรูปแบบ หรือ Copy ทั้งหมดได้เลย
- ตรวจ Schema Type อัตโนมัติ (NewsArticle, Article, HowTo, FAQPage)
- GEO Entities สำหรับให้ AI อ้างอิง

## Deploy บน GitHub + Vercel

### ขั้นที่ 1 — GitHub
1. สร้าง repo ใหม่ เช่น `seo-geo-generator`
2. GitHub Desktop → Clone → วาง `index.html` และ `README.md`
3. Commit → Push

### ขั้นที่ 2 — Vercel
1. vercel.com → Add New Project → เลือก repo
2. Deploy → ได้ URL เช่น `seo-geo-generator.vercel.app`
